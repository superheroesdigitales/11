📚 Estructura de la Diplomatura

Curso 1 – “Domina la Programación con la Mente”

Duración: 2 meses

Objetivo: Usar IA para programar placas Arduino UNO y ESP8266.

  • Fundamentos de programación visual
  • Entrenamiento de la IA para que programe
  • Conexión a placas reales a distancia
  • Proyectos con luces, motores y sensores

Curso 2 – “Crea Tableros de Comando con IA”

Duración: 2 meses

Objetivo: Diseñar interfaces web con IA para controlar objetos reales.

  • Diseño visual de botones y sliders
  • Código HTML, CSS y JS generado por IA
  • Control remoto de objetos reales
  • Automatización con estilo profesional

Curso 3 – “Construye Servidores Web Embebidos”

Duración: 2 meses

Objetivo: Convertir placas Arduino/ESP en servidores web inteligentes.

  • Servidores web embebidos sin IoT
  • Control desde la web sin intermediarios
  • Proyectos finales integrados
